IValve FlowCalibrationInterface.ICalibration.anlernen ( int  timeout)

Erzeugt ein neues Objekt mit der Schnittstelle IValve anhand eines empfangenen Anlerntelegramms. Dieses Ventil wird intern automatisch aktiviert und muss beim Löschen deaktiviert werden.

Parameter:
timeoutDie Zeit in Millisekunden in der nach einem Anlerntelegramm eines Ventils gesucht wird.
Rückgabe:
Ein IValve Objekt mit der Schnittstelle IValve.
Ausnahmebehandlung:
CalibrationExceptionError-Codes

NOT_INITIALIZED:
Der EnOcean-Empfänger wurde nicht initialisiert.

NO_LEARN_MESSAGE:
In der Angegebenen Zeit wurde kein Anlerntelegramm empfangen.

VALVE_ALREADY_LEARNED:
Es wurde versucht ein Ventil anzulernen, welches bereits angelernt wurde.

NO_FREE_IDS:
Es sind bereits 128 Ventile angelernt. Weitere Ventile können nicht angelernt werden.
